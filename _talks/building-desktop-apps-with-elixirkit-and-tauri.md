In this talk I'll present building cross-platform desktop applications with ElixirKit and Tauri. Tauri is a framework for creating small, fast, and secure cross-platform applicat
ions using Web frontend technologies and/or Rust. ElixirKit started out as Elixir and Swift/C# bridge for Livebook Desktop and in the latest iteration has been re-designed to use
 Rust -- a perfect fit for Tauri. We will also cover _distributing_ desktop applications, including code-signing and automatic software updates.
