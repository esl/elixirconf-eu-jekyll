Harmonizing Information: using Elixir for Data Sonification

Speaker:
Carlos Rodrigues


Abstract:

Explore the symphony of possibilities as we delve into the realm of data sonification using Elixir. This talk will guide you through the process of transforming raw data into an audio experience, using the insightful statistics from Lisbon’s open data. Uncover the art and science behind turning data into sound and discover how this approach can offer new insights and perspectives in data analysis.

Key takeaways:

  • Learn the art and science of data sonification and its potential applications in data analysis.
  • Discover creative approaches to making data more accessible and engaging through auditory experiences.
  • Explore tools and techniques for implementing data sonification using Elixir.

Audience:

  • This talk is suitable for data enthusiasts, developers, musicians and anyone interested in exploring innovative ways to interact with and understand data. No prior knowledge of data sonification is required, making it accessible to a broad audience.

Level: Intermediate

Tags: Sonification, Data, Insight

Format: in-person
